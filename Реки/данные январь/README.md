# Папка «данные январь»

## Что здесь лежит

| Путь | Тип | Использование |
|------|-----|----------------|
| `Данные.xlsx` | 14 листов | Паводки, метео, реестры, **уровни** (лист 6) |
| `Уровни_воды.xlsx` | Сводная таблица | Только справка, не для ML-ряда |
| `Кодбук/` | Описание полей | Документация |
| `ДОП/` | ГТС, PDF | Справка |

## Экспорт в нормальные CSV

Из корня проекта:

```powershell
python scripts/export_january_data.py
```

Результат:

- `export/` (эта папка)
- `data/reki/january/`

Главный файл для уровней воды: **`dannie_water_levels_detail.csv`**

См. также `export/README.txt` и `manifest.json` после экспорта.

## В приложении

1. **Река и модель** → кнопка **Экспорт «данные январь» → CSV**
2. **Собрать все посты реки**
3. **Аналитика** → вкладка **Уровень воды**
4. **Прогноз** → ряд **Уровень воды** или **Совместный**, цель `water_level_cm`

Полный свод: `../../PROJECT_SUMMARY.md`
